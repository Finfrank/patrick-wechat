# patrick-wechat
⭐️🐟 questionnaire wechat app built with taro, taro-ui and heart❤️.

## Develop

```bash
# install taro cli
yarn global add @tarojs/cli

# build and watch
yarn run dev:weapp
```

## Update

```bash
# update taro
taro update self

# update project
taro update project
```

## Deploy

```bash
# build and compress
yarn run build:weapp
```

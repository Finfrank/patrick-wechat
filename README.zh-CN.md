# patrick-wechat
⭐️🐟 基于taro, taro-ui的小程序问卷app❤️.

## 开发

```bash
# install taro cli
yarn global add @tarojs/cli

# build and watch
yarn run dev:weapp
```

## 更新

```bash
# 更新 taro
taro update self

# 更新项目
taro update project
```

## 部署

```bash
# 构建、压缩
yarn run build:weapp
```
